
- `species number`・`樹種`は特徴量に**絶対使わない**
- これはNIRの**calibration transfer問題**（確立した研究領域）

## 現在の最良提出

**LB: 17.200**（2026-06-24確認）  
ファイル: `submissions/submission_pls3_5500up.csv`  
手法: **SNV（全波長）→ 5500 cm⁻¹以上を選択 → PLS(n=3) → clip[0.84, 298.58]**

### 物理的根拠

北海道大学 O-7論文（literature/O-7.pdf）より：
- 5200 cm⁻¹付近（O-Hの伸縮+変角の結合音）が高含水率で**飽和**する
- 飽和帯域を除去することで種間転移性が大幅向上
- 5500 cm⁻¹カットオフが最良（LOSO median 13.78）

名古屋大学 稲垣・土川グループ論文（literature/kaken.nii.ac.jp_16K07805seika.pdf）より：
- 木材の散乱係数は吸収係数の10³〜10⁴倍（SNVだけでは補正不完全）
- 細胞壁割合・最大細胞径が散乱を決定（針葉樹/環孔材/散孔材で別回帰）
- 根本的解決は空間分解分光法（SRS）だが Bruker MATRIX-F には非搭載

## 確定した失敗パターン（再現禁止）

| パターン | 結果 | 理由 |
|---------|------|------|
| 樹種特徴量 / random KFold | LB大幅悪化 | 完全リーク |
| GroupKFold/LOSOでの帯域・ハイパーパラメータ最適化 | LB悪化 | train樹種への過適応 |
| EPO（種平均/乾燥曲線） | 悪化 | 水分信号まで除去 |
| 非線形モデル（SVR, LightGBM） | 悪化 | 訓練樹種に過剰適合 |
| MSC/EMSC（高度散乱補正） | 悪化 | SNVが既に最善 |
| 帯域アンサンブル（5400+5500） | LOSO改善→LB悪化 | LOSO改善がLBに反映されない |
| 上限カットオフ（5500-8500等） | 悪化 | 有用情報の除去 |
| SG1前処理 | meanは改善、medianは悪化 | LBはmedianと相関 |

## CV/LBの法則（実証済み）

- GroupKFold CVはLBより約−8程度低め（スギ35%=針葉樹優位のため）
- **LOSOメジアンがLBと最も相関**（meanよりmedianを信頼）
- LOSOで改善してもLBが改善しないケースが複数回確認（LOSO改善を盲信しない）
